# mvvmFX-plugin
Plugin for Intellij IDEA to create a mvvmFX Project. [mvvmFX](https://github.com/sialcasa/mvvmFX) is an application framework for JavaFX that supports the **Model-View-ViewModel** pattern.

## How to install

At the moment the plugin isn't available in the Jetbrains plugin repository. Instead you have to install it manually.

1. Download the JAR file from the [releases section](https://github.com/on-delete/mvvmFX-IntelliJ-Plugin/releases).
2. In IntelliJ IDEA open the settings menu and go to the plugins section.
3. Click "Install plugin from disk..." and choose the downloaded JAR file.
4. Restart to enable the plugin
